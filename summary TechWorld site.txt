A simple PLP site was created associated with 3 tech categories - Laptops, Keyboards, Headsets.

Everything from the requirements like sorting, filtering, is implemented in
separtate sortfilter.js file.

Product counter and creation of the products and product cards is in script.js
Data of how much products are generated and with what fields is in data.js

Each product has it's image and name;
Brand filter names are received dynamically from each product model;
Navigation menu is sticky as required;
Data is generated by a js function which creates for example laptop model from an array with models like 
Macbook, Thinkpad, Asus TUF and appends a number to the name (ex. Macbook 1).
There is 45 products of each category so page loads first 20, next load more click loads next 20 and 
final click loads last 5.
Also when filtering by price range cheapest products come sorted first.
Responsiveness for devices is done by bootstrap integrated specifiers.
Add to cart button makes toast notification.
Each category has unique description connected to its title.
When category is changed through menu the description and title are changed dynamically above.

Techologies used are:
- HTML
- JS
- CSS bootstrap

Challenges encountered during development were:

Sorting alphabetically cause products don't have unique names so I first compare by model name then 
by the appended number in the name.
Responsiveness for devices was a bit challenging too.
And filtering by brand name was also challenging cause there wasn't a brand field in the 
products at first and there was no way to filter it at first.
